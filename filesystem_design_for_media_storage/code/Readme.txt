Project title : File system for images in android
Team member:  Shivam Agarwal,  Dipesh Palod,  Shubham Bansal

-----------------------------------------------------------------------------------------------------------------------------------------

What must be in your zip folder:

1. project-report
2. readme.tex
3. Source folder

------------------------------------------------------------------------------------------------------------------------------------------

Softwares required:
(if you want to run it on emulator)

1. java-eclipse 
2. ADT plugin 17.0
3. android SDK 17.0
4. android platform 4.03 (with API 15)
5. Android NDk 17.0   (not required since libs is already compiled in source)
   (optional) 


-----------------------------------------------------------------------------------------------------------------------------------------

this program is implemented on sd-card.
1.  in sd card you have to create a new folder "storage"
2.  in storage folder,  you have to create a 2GB file using dd command.

How to run program:

1.  extract zip folder.   (use option:  Extract to "y10uc107_y10uc304_y10uc315") 
2.  now in folder "y10uc107_y10uc304_y10uc315",  extract "Source folder to your eclipse workspace folder"
3.   now in eclipse go to new android project
4.   select project from existing source
5.   now open our source from workspace
6.	  create new AVD with sd card size greater than or equal to 2 gb.
7.	 create storage folder. and create 2 gb file using dd in storage folder
8.   run
